# 🚀 Ad-Driven Lead Generation Website

A lightweight, high-conversion landing page and multi-page website built with semantic HTML5 and clean CSS3. This project is optimized for digital marketing campaigns, fast loading speeds, and seamless lead capture.

## 🎯 Purpose & Features

The primary goal of this website is to turn ad traffic into high-quality business leads. 

*   **Conversion-Focused Layout:** Designed with clear Value Propositions and prominent Calls-to-Action (CTAs) above the fold.
*   **Formspree Integration:** The "Contact Us" page uses a serverless form processing integration via Formspree, sending user inquiries directly to your email without requiring a backend.
*   **Fully Responsive:** Optimized for mobile, tablet, and desktop viewports to ensure zero drop-offs from mobile ad traffic.
*   **SEO & Performance Friendly:** Built with vanilla HTML/CSS to ensure lightning-fast load times, boosting ad quality scores (Google Ads/Meta Ads).
